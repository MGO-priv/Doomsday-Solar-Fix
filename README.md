Doomsday Solar Fix
------------------

Portable offline web app for estimating latitude and longitude, using compass and chronometer, for sea navigation purposes. No internet nor sextant is required. 

Accuracy is strongly related to latitude and date.

If you're on a yacht, it is recommended to use compass correction for most accurate results.

It uses a built-in JavaScript solar solver (no Skyfield required) and optional magnetic correction based on World Magnetic Model. Currently used model is valid until 2029.

No guarantee. Use at your own risk.

Built using GPT-5.4 AI (idea based on halucination of protein-based intelligence).
