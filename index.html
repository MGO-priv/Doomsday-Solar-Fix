Highest-confidence observation guidance
  • Best chances are usually when the Sun is far from the equinoxes, so around the solstice seasons.
  • Confidence is usually strongest at higher solar declination and weakest when the azimuth is near 90° or 270°.
  • Sea horizon, true-north azimuth reference, and accurate UTC time all matter.
  • Typical stronger cases are farther poleward within the selected hemisphere, especially when the sunrise/sunset point is visibly displaced from due east/west.
Put world-map.png, app.js, and WMM.COF in the same folder as this HTML file. If the browser blocks local file loading, use the WMM.COF file picker. The magnetic correction feature uses World Magnetic Model.
